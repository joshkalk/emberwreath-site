# Image Naming Convention

Use this convention for website-ready public image assets in `src/assets/images/`.

## General rules

- Use lowercase kebab-case.
- Use `.webp` for website-ready public assets unless there is a specific reason to keep `.png`.
- Keep original full-size source art outside the repo.
- Filenames should describe type and usage, not long narrative details.

## Map naming

Use:

- `emberwreath-map.webp`

Optional future variants:

- `emberwreath-map-labeled.webp`
- `emberwreath-map-player.webp`
- `emberwreath-map-small.webp`

## Species naming

For each species, use:

- `{species}-routine.webp`
- `{species}-class-{class}.webp`
- `{species}-class-{class}.webp`
- `{species}-reference.webp`

Examples:

- `ferren-routine.webp`
- `ferren-class-cleric.webp`
- `ferren-class-ranger.webp`
- `ferren-reference.webp`
- `axali-routine.webp`
- `axali-class-bard.webp`
- `axali-class-rogue.webp`
- `axali-reference.webp`

The routine image filename should not include the specific action. For example, use `aumari-routine.webp` rather than `aumari-forage.webp`.

## Island naming

For each island or region, use:

- `{island}-hero.webp`

Examples:

- `breadbasket-hero.webp`
- `redwind-hero.webp`
- `reedmere-hero.webp`
- `veiled-reefs-hero.webp`
- `western-forbidden-caldera-sea-hero.webp`

## Suggested future usage

- Routine species image: species overview / card.
- Class images: full species page adventurer examples.
- Reference image: full species page reference section.
- Island hero image: island overview page.
- Map image: start-here / atlas pages.
